Certains client (VIP) requièrent une chambre de luxe. Pour augmenter le niveau de la chambre, il faut qu'elle soit vide, il suffit alors de cliquer dessus et de choisir l'option **Améliorer la chambre**. Chaque amélioration a un coût, et demande un délai de construction avant d'être utilisable.

Pour **placer** un client dans une chambre, il vous suffit tout simplement de cliquer sur lui dans la file d'attente et ensuite cliquer sur une chambre libre.

Pour **déplacer** un client dans une autre chambre, vous devez cliquer sur la chambre où se trouve le client, cliquez sur “Déplacer ce client” et ensuite vous cliquez sur une chambre libre.
*Notez qu'il n'est pas possible de permuter deux clients sans chambre libre.*

Note: Si vous déplacez un client celui-ci recevra un malus de -2 points en tant que “déménagement”. Si vous déménagez à nouveau votre client, le malus restera à -2. Ce malus peut être réduit à -1 en recherchant la **Faculté d'adaptation**.

Pour **placer** un client dans une autre salle, il vous suffit tout simplement de cliquer sur lui dans la file d'attente, ou depuis sa chambre et ensuite de cliquer sur la salle ou vous souhaitez le transférer.

Note: Si vous déplacez un client vers l'institut du bonheur, tous les clients présents dans l'hôtel, qu'ils soient dans une chambre, ou dans une file d'attente recevront un malus de -1. Il est donc préférable d'envoyer les clients à l'institut lorsque votre hôtel est vide ou presque.
La recherche **Cellule capitonnée** permet de réduire la probabilité de voir ce malus arriver.

Idée: les “préfixes” signalent le type de client:

- Pour une sorcère : Circé-, Divina-, Magi-, Malé-, Maléfi-, Mandra-, Maudi-, Méfis-, Paga-, Sabba-, Sorbe-, Sorce-, Sorti- ou Stry-.

- Pou un capitaliste : Biz-, Boursi-, Capita-, Euro-, Finan-, Fortu-, Frikou-, Grisbi-, Luxu-, Moné-, Pacto-, Richi- ou Tréso-.

- Pour un pyromane : Ardento-, Charbo-, Chau-, Combu-, Flamboi-, Flame-, Infla-, Napal-, Orifla-, Pyro-, Tourbo- ou Volca-.

- Pour une abomination : Abomina-, Bizaro-, Dégueu-, Franken-, Hiddo-, Horri-, Laide-, Monstru- ou Ostro-.

- Pour un zombie : Cada-, Frela-, Momi-, Mor-, Morbi-, Nécro-, Pesti-, Pourri-, Puru-, Putré-, Putri-, Véro- ou Zombi-.

- Pour un hypno-chat : Aéro-, Ciele-, Démo-, Diabo-, Fauvi-, Féli-, Hypno-, Lévi-, Migno-, Minou-, Neko-, Neuro-, Sata- ou Vol-.

- Pour un blob fauché : Amido-, Blobo-, Col-, Gélati-, Géli-, Gluti-, Gluo-, Pata- ou Visquo-.

- Pour un poisson sourd : Algue-, Amphi-, Aqua-, Colina-, Maré-, Nauti-, Pané-, Poisso-, Saumo- ou Tourto-.

- Pour une hystérique : Crué-, Furi-, Irita-, Mazo-, Pervi-, Sadis-, Sado- ou Vicia-.

- Pour une monstro-plantes : Arbo-, Banbou-, Cactu-, Floru-, Herba-, Liani-, Planta- ou Végé-.

- Pour un fantôme : Imma- ou Polter-.


Les “suffixes” sont communs a touts les clients, on peux trouver les suffixes suivants :
